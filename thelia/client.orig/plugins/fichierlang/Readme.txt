Ce plugin permet de traduire l'interface du site Internet.

Copiez le répertoire lang à la racine de votre installation THELIA (au même niveau que les squelettes)

Vous pouvez ensuite éditez les fichiers lang1.php, lang2.php pour traduire les informations.

Ex :

Dans votre squelette, vous pouvez écrire ::compte::

lang1.php contient :
'compte' => 'Mon compte'

et lang2.php contient :
'compte' => 'My account'

Si le site est affiché en français ::compte:: sera remplacé par "Mon compte", en anglais par "My account"
