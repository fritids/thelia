Ce plugin vous permettra de valider l'inscription d'un client avant qu'il puisse se connecter.

Veuillez simplement glisser le répertoire validcli dans le dossier client/plugins de votre Thelia.
Vous pouvez aussi utiliser l'assistant d'import de plugins dans Configuration/Gestion des plugins.
