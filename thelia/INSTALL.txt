THELIA est écrit en PHP.

Pour installer Thelia, il suffit de pointer votre navigateur WEB à l'adresse suivante :
http://votreserveurweb/thelia/install/

Remplacez "thelia" par votre répertoire d'installation bien sûr.

une fois l'installation terminée, supprimez le répertoire install.

Copiez ensuite les fichiers du répertoire template à la racine de votre installation si vous souhaitez découvrir le template de base.
